Le Socket

Una socket è un endpoint per la comunicazione tra due macchine in una rete. Approfondisci nel glossario.

Le socket possono essere di due tipi principali: Stream (es. TCP) e Datagram (es. UDP).

Il processo di comunicazione con le socket prevede un client che invia richieste a un server e il server che risponde tramite una connessione stabilita.

Nicolo Malaman, Classe 5F
